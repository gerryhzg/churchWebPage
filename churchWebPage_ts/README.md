# 达拉斯第一浸信会华人团契

Astro + TypeScript rewrite of the static WordPress export from `churchWebPage_wp`.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

The old WordPress export is intentionally not required at runtime. Uploaded media is served from `public/wp-content/uploads` so existing image, document, and video URLs continue to work.
